# ☕ Sistema de Cadastro e Reajuste Salarial (Java)

Este projeto é uma aplicação de console desenvolvida para consolidar conceitos fundamentais de **Programação Orientada a Objetos (POO)** e manipulação de coleções em Java.

## 🚀 Tecnologias Utilizadas
*   **Linguagem:** Java (JDK 21)
*   **Paradigma:** Orientação a Objetos
*   **Estrutura de Dados:** Listas Dinâmicas (`ArrayList`)

## 🛠️ Funcionalidades
- [x] Cadastro de funcionários com ID único, Nome e Salário.
- [x] Validação de ID para evitar duplicidade.
- [x] Busca de funcionário por ID.
- [x] Aplicação de reajuste salarial baseado em porcentagem.
- [x] Listagem completa de funcionários registrados.

## 🧠 Conceitos de Engenharia de Software Aplicados
Durante o desenvolvimento, apliquei conceitos essenciais para um código limpo e organizado:
*   **Encapsulamento:** Uso de atributos privados com Getters e Setters.
*   **Modularização:** Separação de responsabilidades entre as classes de entidade e a classe principal.
*   **Expressões Lambda:** Utilização de Stream API e predicados para busca eficiente de elementos na lista.

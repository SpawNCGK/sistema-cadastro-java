# Sistema de Cadastro em Java

Projeto simples para praticar conceitos de programação orientada a objetos.

## Funcionalidades

- Cadastro de cliente
- Listagem de clientes

## Estrutura

model → entidades do sistema  
repository → armazenamento de dados  
service → regras de negócio  
main → execução do programa

## Tecnologias

- Java
- Git
